Inview is a binary file viewer (and editor) for Windows. It opens files in 
hexademical, character, and decimal value. Lines are 80 characters wide.

This is a modified copy of the original Inview from computers class:
Programming Applications 1, Class 2, April 3, 2001

Requires:

Microsoft Visual Basic 6 runtime
